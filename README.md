# Camisetas Icónicas

Catálogo de camisetas retro con búsqueda en tiempo real.

## Demo
https://camisetasiconicas.netlify.app/

## Tecnologías
- JavaScript vanilla
- Google Apps Script (backend)
- Google Sheets (base de datos)
- Netlify (hosting)

## Características
- 54 productos
- Búsqueda en tiempo real
- Filtros por equipo
- WhatsApp para consultas
- Carga en 5 segundos
